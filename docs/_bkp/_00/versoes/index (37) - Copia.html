§ manifesto 02
01

A IA como lanterna, não como destino

Comecei sem saber escrever uma linha de CSS. A transição não foi planejada como um curso — foi uma série de perguntas que pediram ferramentas novas para serem respondidas.

O que a IA tornou possível não foi apenas "gerar código mais rápido". Foi poder levar uma ideia que existia só na cabeça até um protótipo funcional em horas — e então entender, depurar, refinar.

02

Versionar é contar a história

O GitHub aqui não é só repositório — é diário de desenvolvimento. Cada commit tem intenção. Cada pasta tem um README que contextualiza por que aquilo existe.
